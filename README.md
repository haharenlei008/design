# AI 宝宝投影灯首页原型

这是一个面向 AI 宝宝投影灯的静态首页交互原型，包含设备在线态、设备离线态、模式切换、随机播放、AI 生成音频、语音留言、灯光与旋转等核心首页能力。

## 页面入口

- `index.html`：设备在线首页，支持完整交互。
- `offline.html`：设备离线首页，设备相关操作保持禁用。

## 核心功能

- 四种模式切换：故事模式、儿歌模式、睡眠模式、亲子模式。
- 模式切换动效：背景、标题、副标题、随机播放文案和氛围粒子同步变化。
- 随机播放：可换一个主题、试听主题片段，并发送到设备播放。
- AI 生成：输入生成要求后，先进入约 5 秒准备阶段，再模拟音频流式传输。
- AI 试听：流式传输过程中可试听片段，传输完成后可完整试听。
- 设备发送：音频流传输完成后，可发送给设备播放并更新首页播放卡片。
- 语音留言：长按录音，完成后可试听并发送到设备播放。
- 模式默认效果：故事、儿歌、睡眠、亲子模式会自动匹配灯光模式、亮度和旋转速度。
- 快捷控制：灯光设置、语音留言、旋转设置、停止播放、开启/关闭场景效果、设备切换、头像侧边菜单。

## 本地预览

可以直接打开 HTML 文件，也可以启动一个本地静态服务器：

```bash
python3 -m http.server 4173
```

然后访问：

```text
http://127.0.0.1:4173/index.html
http://127.0.0.1:4173/offline.html
```

## 开发说明

- 当前项目是纯前端静态原型，不依赖构建工具。
- AI 生成、随机播放、语音发送均为前端模拟，不连接真实后端或真实设备。
- AI 音频通过前端生成的临时 WAV Blob 模拟，便于演示试听和发送流程。
- `.codex/` 已加入 `.gitignore`，不会提交本地 Codex 工作目录。

## 验证

基础语法检查：

```bash
node --check app.js
```

建议手动验证：

- 在线首页 AI 生成：空输入提示、5 秒准备、流式进度、试听、发送成功。
- 随机播放：换一个、试听、发送成功。
- 语音留言：长按录音、试听、发送。
- 灯光与旋转：切换灯光模式、调节执行速度和整体亮度、关闭灯光、切换或停止投影旋转。
- 离线首页：设备操作按钮保持不可用，AI 立即生成不可点击。

## 模式卡片文案与本地化规则

- 模式卡片标题按“两行内可读”设计，建议常规文案控制在 4~8 个中文字符（或 8~16 个英文字符）。
- 极端文案（例如活动态、多语言替换）允许拉长，但应限制在最多两行显示，并通过省略截断避免遮挡图案与选中态光效。
- 为保证深浅背景下可读性，卡片标题统一使用最小字号（不低于 24px）、加粗字重、文字阴影与底部遮罩的组合对比策略。
- 本地化建议：
  - 中文优先短词（如“故事模式”），避免超过 10 个汉字。
  - 英文/拉丁语种优先 2~3 词短语，避免超过 18 个字符（含空格）。
  - 超长语言（如德语复合词）需在翻译阶段给出短标签版本。

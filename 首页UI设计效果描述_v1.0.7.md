# AI投影灯 首页UI设计效果描述

**文档版本**：v1.0.7  
**编制日期**：2026年5月

---

## 1. 页面整体氛围

**不是控制台，是温馨小舞台。**

打开App的第一感受应该是——"哦，好温暖"，而不是"哦，好多按钮"。

**核心视觉公式**：
```
温馨小舞台 = 深紫蓝夜色底 + 暖金色投影灯光芒（视觉中心）+ 一行4色模式卡片 + 暖黄星光点缀
```

**氛围关键词**：温暖、安心、童趣、简洁、有呼吸感

---

## 2. 布局节奏

疏密有致，中间松、两头轻：

```
顶栏（极简）→ 投影灯主视觉（留白+光芒）→ 一行4个模式卡片 → 两行操作栏 → 快捷栏
    轻              重（视觉中心）              中            中          轻
```

**页面留白**：投影灯区域上方和下方各留出舒适间距，不要密密麻麻堆满内容。

---

## 3. 各区域视觉详述

### 3.1 顶栏（极简）

**视觉描述**：
- 高度 36px，完全透明背景（与页面底色融合）
- 左侧：设备名称（H4层级，白色 `#FFFFFF`，字重medium）+ 绿色在线圆点（🟢8px发光）+ "设备在线"小字（10px，次要色）
- 右侧：用户头像（28px圆形卡通头像，带发光边框 `#FCD34D` glow效果），点击弹出右侧菜单
- 无背景、无模糊、无分隔线，几乎"不存在"的顶栏

### 3.2 投影灯主视觉区

**这是整个页面的灵魂。**

**视觉描述**：
- 区域高度约 180px，居中展示
- 中心：兔子投影灯形象插图（约120px高）
  - 兔耳：暖粉色半透明质感
  - 灯罩：磨砂半透明，有镂空星芒/行星/月牙图案，暖金色灯光从镂空处透出
  - 底座：米白色圆润弧形，中间嵌蓝色发光环"AI"触控按钮
  - 周围：暖金色光芒辐射（径向渐变 `rgba(252,211,77,0.15)` → 透明），模拟灯光照亮周围
- 投影灯上方/背后：细微星空光点散落（3-5颗暖黄微光点），模拟投影投射效果
- 投影灯下方：模式标题（带书本图标 + "故事模式"，圆润暖金色字体，H3层级）
- 模式标题下方：副标题小字"今晚，陪宝宝听一个温暖故事"（Body层级，白色60%透明度），随模式变化
- 模式标题右侧：两个圆形功能按钮
  - 💡灯光按钮：圆形（40px），带灯泡图标+文字"灯光"
  - 🔇静音按钮：圆形（40px），带喇叭图标+文字"静音"

**光芒呼吸动效**：已连接状态下，暖金色光芒4秒周期微微扩散收缩，像呼吸一样，传递"我在陪伴"的感觉。

**4态视觉**：

| 状态 | 投影灯表现 |
|------|-----------|
| 未连接 | 投影灯灰暗，无光芒，无星光，"未连接"文字 |
| 连接中 | 投影灯半亮，光芒若隐若现，🔄旋转 |
| 已连接 | 投影灯暖金色光芒绽放，星空光点散落，模式标题+副标题清晰，灯光/静音按钮可操作 |
| 连接失败 | 投影灯灰暗，⚠️红色警告图标微光 |

### 3.3 模式选择区（一行4个紧凑卡片）

**视觉描述**：
- 4个卡片一行紧凑排列，卡片尺寸：约 屏幕宽度22% × 80px，圆角12px
- 卡片间距紧凑，整体居中排列
- 每个卡片有独立的色调风格和插画图案

**单张卡片视觉**：

| 模式 | 色调 | 插画图案 | 视觉表现 |
|------|------|---------|---------|
| 📖 故事模式 | 橙黄色调 | 打开的故事书与城堡图案 | 温暖的故事场景插画 |
| 🎵 儿歌模式 | 粉色调 | 音符图案 | 欢快的音乐场景插画 |
| 🌙 睡眠模式 | 深蓝色调 | 月牙图案 | 静谧的睡眠场景插画 |
| 👨‍👧 亲子模式 | 绿色调 | 亲子互动图案 | 温馨的亲子场景插画 |

**卡片结构**：
- 插画图案占卡片80%，插画风格（非照片），圆润可爱
- 图案下方20%：模式名称，模式色调浅色底，白色文字居中

**交互态**：

| 状态 | 视觉 |
|------|------|
| 默认 | 插画图案完整展示，无边框 |
| 运行中 | 高亮发光边框（暖黄色 `#FCD34D` glow效果），其他卡片无边框 |
| 按下 | 缩放96% |

### 3.4 操作栏

**视觉描述**：简洁，两行设计

**第一行：随机播放**（高度 44px）
- 左侧：星芒图标（✨，暖黄色）+ 主文字"选择一个模式开始吧"（Body层级，白色）+ 副文字"点击可随机播放一个内容"（小字，次要色）
- 右侧：圆形随机播放按钮（48px），带双向箭头图标+文字"随机播放"，深色底白字

**第二行：AI生成**（高度 44px，间距12px）
- 左侧：星芒图标（✨，暖黄色）+ 主文字"想听什么故事？"（Body层级，白色）
- 右侧：圆形AI生成按钮（56px），紫橙渐变背景（`#8B5CF6 → #F97316`）+ 文字"立即生成"，白色

### 3.5 快捷控制栏

**视觉描述**：
- 固定页面底部，高度 52px
- 背景：`rgba(0,0,0,0.3)` + 模糊
- 4个圆形按钮（44px）等距排列，半透明深色底：
  - 🔆投影亮度：太阳图标 + 文字"投影亮度"，暖黄色图标
  - 🔊音量：喇叭图标 + 文字"音量"，品牌青色图标
  - 🌙夜灯：月牙图标 + 文字"夜灯"，暖黄色图标（开启时高亮发光边框）
  - 🎤语音：麦克风图标 + 文字"语音"，品牌青色边框
- 点击亮度/音量弹出环形滑块（radial slider），用完自动收起
- 底部适配iOS安全区

### 3.6 顶栏头像菜单

**视觉描述**：
- 从右侧滑入的面板，宽度 240px，占屏幕约65%
- 深色毛玻璃背景 + 暖黄细边框
- 顶部：用户头像（64px，圆形卡通，带暖黄发光边框）+ 昵称
- 菜单项列表，每项 48px高：
  - 💬对话（有未读时右侧暖黄光点）
  - 👤我的
  - ⚙️设置
  - ❓帮助
- 点击菜单项跳转对应页面，面板自动关闭
- 点击面板外区域关闭面板

## 4. 首屏视觉总结

**视觉流**：

```
[顶栏] 宝宝投影灯 ●设备在线               [头像🖼️]
                                            
              ✨  ✨                        ← 星光点缀
           ✨  🐰投影灯  ✨                   ← 视觉中心，暖金色光芒
              ☀️ 光芒 ☀️
           📖 故事模式                       ← 模式标题（暖金色）
           今晚，陪宝宝听一个温暖故事          ← 副标题
        [💡灯光]  [🔇静音]                   ← 圆形按钮

    [📖故事] [🎵儿歌] [🌙睡眠] [👨‍👧亲子]      ← 一行4个紧凑卡片

    [✨选择一个模式开始吧]        [🔄随机播放]
    [✨想听什么故事？]        [✨立即生成]

    [☀️投影亮度] [🔊音量] [🌙夜灯] [🎤语音]   ← 快捷控制
```

**整体视觉印象**：
暗调夜色中，一只暖金色发光的兔子投影灯安静地亮着，周围散落几点星光。下面一行四个彩色小卡片像节目单，每个都有自己的色调和可爱图案。旁边是随机播放和AI生成的入口。底栏四个圆圆的小按钮等着你调。温暖、安心、不复杂。

## 5. 背景与氛围

- 深紫蓝渐变 `#0f0c29 → #1a1040 → #24243e`
- 投影灯区域：叠加暖金色径向渐变光芒（已连接时）
- 极淡星光粒子，不干扰阅读
- 页面内边距 20px
- 一屏展示，无需滚动

## 6. 动效设计

| 元素 | 动效 | 周期 | 说明 |
|------|------|------|------|
| 投影灯光芒 | 呼吸扩散 | 4s | 暖金色光芒微微扩散收缩，"我在陪伴" |
| 星光粒子 | 缓慢闪烁 | 3-5s随机 | 极淡，偶尔闪烁 |
| 模式卡片切换 | 发光边框淡入 | 300ms | 从无边框到高亮发光边框过渡 |
| AI生成中 | 紫橙脉冲 | 1.5s | 生成按钮内文字+光效 |
| 快捷栏弹窗 | 环形滑块弹出 | 200ms | 从图标位置展开 |
| 头像菜单 | 右侧滑入 | 250ms | ease-out |

---

## 7. 设计原则

| 原则 | 说明 |
|------|------|
| 投影灯是主角 | 一切围绕投影灯展开，它是情感锚点 |
| 信息随用随现 | 亮度/音量不常调，点击才弹出；菜单收进头像 |
| 独立色调卡片 | 每个模式卡片有独特的色调和插画风格，视觉更丰富 |
| 留白是呼吸 | 投影灯区域不堆东西，让光芒有空间 |
| 暖金色是温度 | 所有关键反馈用暖金色传递温暖，不用冷色 |
| 儿童不需要控制台 | 他们需要一盏灯、一个故事、一个拥抱 |

# AI投影灯 首页页面功能设计

**文档版本**：v1.0.7  
**编制日期**：2026年5月

---

## 1. 页面概述

**页面名称**：设备首页  
**优先级**：P0  
**入口**：App启动唯一页面

**核心定位**：温馨的儿童陪伴小舞台——投影灯是舞台主角，模式是场景选择，内容是正在上演的节目。首页要让人一眼感受到"温暖、安心、好玩"，而不是面对一堆控件。

---

## 2. 页面结构

从上到下，视觉节奏疏密有致：

| 区域 | 内容 | 视觉权重 |
|------|------|---------|
| 顶栏 | 设备名+在线状态（含"设备在线"文字标签）+ 用户头像（发光边框，点击展开：对话/我的/设置/帮助） | 轻，半透明 |
| 投影灯主视觉 | 投影灯形象 + 模式标题+副标题 + 灯光/静音圆形按钮 | 重，页面视觉中心 |
| 模式选择区 | 4个模式卡片，一行4个紧凑排列，各有独立色调和插画图案 | 中，核心交互 |
| 操作栏 | 两行：随机播放 + AI生成 | 中，实用区 |
| 快捷控制栏 | 投影亮度、音量、夜灯、语音 | 轻，固定底部 |

---

## 3. 顶栏（极简）

| 元素 | 说明 |
|------|------|
| 设备名称 | 左侧，默认"宝宝投影灯"，白色，字重medium |
| 在线状态 | 设备名右侧，绿色在线圆点（🟢8px发光），圆点下方有"设备在线"小字 |
| 用户头像 | 右侧，圆形卡通头像（28px），带发光边框，点击弹出菜单：💬对话、👤我的、⚙️设置、❓帮助 |

**设计意图**：顶栏只保留最必要的信息，对话/我的/设置/帮助收纳进头像菜单，避免顶栏图标拥挤。头像采用发光边框风格，与整体温馨氛围呼应。

## 4. 投影灯主视觉区

### 4.1 核心元素

| 元素 | 说明 |
|------|------|
| 投影灯形象 | 页面视觉中心，兔子投影灯插图（约120px高），灯罩有镂空星芒/行星/月牙图案，暖金色灯光从镂空处透出 |
| 底座 | 米白色圆润弧形，中间嵌蓝色发光环"AI"触控按钮 |
| 模式标题 | 投影灯下方，带书本图标 + "故事模式"（圆润暖金色字体），H3层级 |
| 副标题 | 模式标题下方，小字"今晚，陪宝宝听一个温暖故事"，随模式变化 |
| 💡灯光按钮 | 标题右侧，圆形按钮，带灯泡图标+文字标签"灯光" |
| 🔇静音按钮 | 灯光按钮旁，圆形按钮，带喇叭图标+文字标签"静音"（不是小图标，是带文字标签的圆形按钮） |

### 4.2 设备状态4态

| 状态 | 视觉表现 |
|------|---------|
| 未连接 | 投影灯灰暗 + "未连接设备" + [扫描配网] [模拟体验] |
| 连接中 | 投影灯半亮 + 🔄旋转 + "正在连接..." |
| 已连接 | 投影灯暖黄光芒 + 星空光晕 + 模式标题+副标题 + 灯光/静音按钮 |
| 连接失败 | 投影灯灰暗 + ⚠️ + "连接失败" + [重新连接] |

**设计意图**：把设备状态和投影灯形象融合，投影灯本身就是最好的状态指示——亮着就是在线，暗了就是离线，不需要额外的信息卡片。灯光和静音采用带文字标签的圆形按钮，操作更直观。

## 5. 模式选择区

### 5.1 四大儿童模式

| 模式 | 色调风格 | 插画图案 | 一句话描述 | 核心能力 |
|------|---------|---------|-----------|---------|
| 📖 故事模式 | 橙黄色调 | 打开的故事书与城堡图案 | 会讲故事的好伙伴 | AI讲故事 + 星空投影 |
| 🎵 儿歌模式 | 粉色调 | 音符图案 | 唱唱跳跳真开心 | 儿歌播放 + 可爱投影 |
| 🌙 睡眠模式 | 深蓝色调 | 月牙图案 | 乖乖入睡好梦来 | 星空投影 + 白噪音 + 渐暗 |
| 👨‍👧 亲子模式 | 绿色调 | 亲子互动图案 | 一起玩真开心 | AI互动 + 投影互动 |

### 5.2 模式卡片结构

4个模式卡片**一行4个紧凑排列**，每个卡片宽度约屏幕22%，高度约80px，间距紧凑。

| 元素 | 说明 |
|------|------|
| 插画图案 | 卡片视觉主体，插画风格图案（不是照片场景图），占卡片80% |
| 模式名称 | 图案下方，模式色调的浅色底，文字居中 |
| 选中态 | 卡片带高亮发光边框（其他卡片无边框高亮） |

### 5.3 模式卡片交互

| 状态 | 表现 |
|------|------|
| 默认态 | 插画图案展示，无边框，圆角12px |
| 选中/运行态 | 高亮发光边框（暖黄色 glow 效果） |
| 按下态 | 缩放96% |
| 禁用态 | 灰度+50%（设备离线） |

### 5.4 模式互斥

- 同一时间只运行一个模式
- 切换时当前模式自动退出
- 播放中切换弹出确认

**设计意图**：一行4个紧凑卡片设计，每个卡片有独立的色调和插画风格，视觉更统一、更具童趣感。选中态采用发光边框替代v1.0.6的底部暖黄横条，视觉更精致。

## 6. 操作栏

### 6.1 区域结构

两行操作栏设计，不显示当前播放内容信息：

**第一行：随机播放**
| 元素 | 说明 |
|------|------|
| 星芒图标 | 左侧，暖黄色小图标 |
| 主文字 | "选择一个模式开始吧" |
| 副文字 | "点击可随机播放一个内容"（小字，次要色） |
| 随机播放按钮 | 右侧，圆形按钮，带双向箭头图标+文字"随机播放" |

**第二行：AI生成**
| 元素 | 说明 |
|------|------|
| 星芒图标 | 左侧，暖黄色小图标 |
| 主文字 | "想听什么故事？" |
| AI生成按钮 | 右侧，圆形按钮，紫橙渐变背景+文字"立即生成" |

### 6.2 操作状态

| 状态 | 展示 |
|------|------|
| 未选择模式 | 第一行："选择一个模式开始吧" |
| 模式已选 | 第一行：随机播放按钮可点击 |
| AI生成中 | "AI正在创作..." + 旋转动画 |

### 6.3 各模式AI生成入口文案

| 模式 | 入口文案 |
|------|---------|
| 📖 故事模式 | "想听什么故事？" |
| 🎵 儿歌模式 | "想听什么样的儿歌？" |
| 🌙 睡眠模式 | "想要什么样的助眠？" |
| 👨‍👧 亲子模式 | "想玩什么？" |

**设计意图**：两行操作栏设计简洁直接，随机播放和AI生成入口清晰。不再显示当前播放内容信息，让页面更简洁，突出核心操作入口。

## 7. 快捷控制栏

固定页面底部，4个圆形图标按钮等距排列：

| 控件 | 图标 | 文字标签 | 交互 | 说明 |
|------|------|---------|------|------|
| 🔆投影亮度 | 太阳图标 | "投影亮度" | 点击弹出环形滑块 | 调节投影亮度 |
| 🔊音量 | 喇叭图标 | "音量" | 点击弹出环形滑块 | 调节音量 |
| 🌙夜灯 | 月牙图标 | "夜灯" | 点击开关 | 开启/关闭夜灯（选中高亮发光边框） |
| 🎤语音 | 麦克风图标 | "语音" | 长按 | 弹出语音面板 |

**设计意图**：用圆形图标按钮替代传统滑块条，视觉更干净。点击后弹出环形滑块调节，用完自动收起。夜灯按钮在开启状态时高亮发光边框。

## 8. 顶栏头像菜单

点击顶栏右侧用户头像（带发光边框），从右侧滑入菜单面板：

| 菜单项 | 说明 |
|--------|------|
| 💬对话 | 跳转AI聊天记录列表 |
| 👤我的 | 跳转个人中心 |
| ⚙️设置 | 跳转设备设置 |
| ❓帮助 | 跳转帮助页 |

**设计意图**：保留v1.0.6的菜单功能，头像视觉改为发光边框风格，与整体设计语言一致。
